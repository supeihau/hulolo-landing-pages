# 彰師小生物一頁式網頁

彰師小生物是提供給彰化師範大學的學生課程評價查詢的平台，收錄1000則以上的評價，且目前有6500以上的使用者，用戶可以使用Line Bot或網站(https://ncuehulolo.idv.tw/)進行全校所有課程評價查詢。
